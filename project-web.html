Project web

De Opdracht

Project web was een opdracht voor Transavia, ze wilden een platform waar reizigers stijloefeningen konden lezen als ze onderweg zijn. Hierbij was het belangrijk dat gebruikers verhalen konden liken, downloaden en konden filteren om zo geschikte verhalen te kunnen vinden om onderweg te lezen.

Het Eindresultaat

Het eindresultaat is responsive website met 99 stijloefeningen die ieder op een andere manier zijn vormgegeven. Hierbij is rekening gehouden met de sfeer van het verhaal. In het ontwerpdocument van het project is verdere informatie te vinden over de inhoud van het project.
